# Quine Collection

> programs that print themselves

## Usage

```bash
# test all quines
./check.sh

# run a quine
python3 py.py
clang c.c -o c && ./c
rustc rs.rs -o rs && ./rs
```

## Languages

`Python` `C` `C++` `Rust` `Go` `Clojure` `Bash` `Zig` `Lua` `Elixir` `OCaml`

## Notes

missing compilers are auto-skipped 

each quine is verified to output itself exactly
